# flutter_monetization_kit

A Flutter plugin that provides a unified and easy-to-use API for Google Mobile Ads and in-app purchases on Android and iOS.

---

## ✨ Features

### Google Mobile Ads
- Banner Ads
- Interstitial Ads
- Rewarded Ads
- App Open Ads (planned)

### In-App Purchases
- One-time products
- Subscriptions
- Restore purchases
- Android (Play Billing)
- iOS (StoreKit)

---

```yaml
dependencies:
  flutter_monetization_kit: ^0.0.1
